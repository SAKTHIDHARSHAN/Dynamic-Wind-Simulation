# Dynamic Wind Simulation in Unity

## Features

- Dynamic wind strength control
- Runtime wind direction control
- Physics-based object interaction
- Wind Zone integration
- Falling leaf particle effects
- Interactive UI controls

## Controls

- Wind Strength Slider
- Wind X Slider
- Wind Z Slider

## Unity Version

2022.3.62f3

## Assets Used

- Nature Manufacturer Environment Pack
